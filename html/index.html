Corganiser is a web-based tool for organising whole-round core cutting during scientific drilling, typically for microbiological analysis. It was written by Ian Marshall from the Center for Geomicrobiology at Aarhus University during and following IODP Expedition 347.

Corganiser is written in Python and makes use of the ReportLab package for rendering PDFs. The source code is available through a GitHub Repository.

Please send questions, suggestions, and other feedback to Ian Marshall (ianpgm at biology.au.dk).
